Czy podany kod jest poprawny składniowo w Pythonie? Jeśli nie, to dlaczego?

x = 2; y = 3;
if (x > y):
    result = x;
else:
    result = y;

    Powyższy segment da się uruchomić, natomiast nie jest on poprawny.
    Nie powinno być ; na końcach wyrażeń, oraz deklaracje x i y powinny być w osobnych liniach

for i in "axby": if ord(i) < 100: print (i)

    Kod nie może być w 1 linijce. brakuje wcięć. powinnien być

for i in "axby":
    if ord(i) < 100:
        print (i)

    ewentualnie:

for i in "axby":
    if ord(i) < 100:print (i)


for i in "axby": print (ord(i) if ord(i) < 100 else i)

    kod działa, chociaż nie powinno być spacji między print a nawiasem
